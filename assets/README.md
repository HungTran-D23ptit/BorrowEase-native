# Thư mục `assets`

Mục đích: chứa hình ảnh, font, icons, và asset tĩnh.

Cách dùng chuẩn:

- `assets/images/` — ảnh, chia theo folder `admin/` và `user/` nếu cần asset riêng.
- `assets/fonts/` — các file font.

Quy tắc:

- Không đặt code trong assets; chỉ chứa tệp tĩnh.
- Nếu có quá nhiều image dành riêng cho admin/user thì sử dụng `assets/admin/` và `assets/user/`.
